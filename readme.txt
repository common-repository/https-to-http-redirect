=== HTTPS to HTTP Redirect ===
Contributors: ratanmia
Stable tag: 1.0
Tested up to: 4.9
Requires at least: 4.0

This plugin able to redirect all https request into https.

== Description ==

This plugin will redirect all HTTPS request to HTTP request. But, You must have installed SSL certificate.

== Installation ==

Install like any other plugin, directly from your plugins page.
